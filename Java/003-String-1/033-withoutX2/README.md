033- Java- String-1
==================

withoutX2
-----------

Problem:  
Given a string, if one or both of the first 2 chars is 'x', return the string without those 'x' chars, and otherwise return the string unchanged. This is a little harder than it looks. 
>
withoutX2("xHi") → "Hi"  
withoutX2("Hxi") → "Hi"  
withoutX2("Hi") → "Hi"
