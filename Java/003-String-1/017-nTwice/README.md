017- Java- String-1
==================

nTwice
-----------


Problem:  
Given a string and an int n, return a string made of the first and last n chars from the string. The string length will be at least n. 
>
nTwice("Hello", 2) → "Helo"  
nTwice("Chocolate", 3) → "Choate"  
nTwice("Chocolate", 1) → "Ce"
